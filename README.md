# Student-Management-System

A simple Student Management System built using C programming and file handling.
This project allows users to add, display, search, and delete student records using a binary file.

Features:

1. Add new student records

2. Display all students

3. Search student by roll number

4. Delete student by roll number

5. Menu-driven console interface

Technologies Used:

1. C Programming Language

2. File Handling (Binary Files)

3. Structures

4. Functions
